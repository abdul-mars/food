Project name: online food delivery system
to run this project.
- extract and rename the folder to 'food' (if its already not the name) and copy the folder to htdocs in your xampp folder
- open create a database and name it food
- import the database (food.sql)
- to open users' page, open your browser and type in the url (localhost/food)
- you will need to log in before you can order food and to log in to users page create new account
- to open admin panel/dashboard type (localhost/food/admin), enter login credentials
